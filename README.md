# Diplomat University Website

This is a Django-based web application for Diplomat University, featuring:
- Multilingual support (UZ, EN, RU)
- University shop for merchandise
- Leadership and Department management
- Student and Employee information system

## Development
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables in `.env`
4. Run migrations: `python manage.py migrate`
5. Start server: `python manage.py runserver`
